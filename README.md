# SeeForest — Random Forest Visualizer

Interactive animated presentation teaching the Random Forest algorithm for a 25-minute Machine Learning Group Seminar.

19 slides covering three parts: Why Random Forest, How It Works, and Mastering RF.

## Quick Start

```bash
npm install
npm run dev
```

## Controls

- **Arrow keys / Space**: Next step
- **Backspace**: Previous step
- **L**: Toggle English / Chinese
- **Ctrl+1-9**: Jump to slide

## Handout

```bash
cd handout
pdflatex handout.tex
pdflatex problemset.tex
```

## Tech Stack

React + TypeScript + Vite + Tailwind CSS + Framer Motion + Three.js + Zustand
